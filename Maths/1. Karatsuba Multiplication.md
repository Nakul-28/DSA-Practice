# Karatsuba Multiplication
Karatsuba Multiplication is an efficient algorithm for multiplying large numbers. It reduces the multiplication of two n-digit numbers to at most three multiplications of n/2-digit numbers, which is faster than the traditional O(n^2) method.

## Algorithm
1. Split the input numbers into two halves.
2. Recursively compute three products:
$$X = 5678  (56=a, 78=b)$$
$$Y = 1234  (12=c, 34=d)$$
   - $z_0 = a * c$ (the product of the lower halves)
   - $z_1 = (a + b) * (c + d)$ (the product of the sums of the halves)
   - $z_2 = b * d$ (the product of the upper halves)
3. Combine the results using the formula:
    - $result = z_0 * 10^{(2*m)} + (z_1 - z_2 - z_0) * 10^m + z_2$
    where m is the number of digits in the halves.


## A Recursive Algorithm

1. Write $X = 10^{n/2} \cdot a + b$ and $Y = 10^{n/2} \cdot c + d$ (where $a$, $b$, $c$, and $d$ are the n/2 digit sequences of the numbers).
2.  $X \cdot Y = (10^{n/2} \cdot a + b)(10^{n/2} \cdot c + d)$ <br>
$$= 10^n ac + 10^{n/2}(ad+bc) + bd$$
3. Recursively $ac,ad,bc,bd$, then compute (Step 2 calc) in the straightforward way. <br>
- Base Case for this algorithm is when the numbers have only one digit, in which case we can directly multiply them.

## Time Complexity
The time complexity of the Karatsuba algorithm is O(n^log2(3)), which is approximately O(n^1.585). This is significantly faster than the traditional O(n^2) method for large numbers.

## Implementation in Java 
```java
public class Karatsuba {
    public static long karatsuba(long x, long y) {
        // Base case for recursion
        if (x < 10 || y < 10) {
            return x * y;
        }

        // Calculate the size of the numbers
        int maxLength = Math.max(Long.toString(x).length(), Long.toString(y).length());
        int halfLength = maxLength / 2;

        // Split the digit sequences in the middle
        long a = x / (long) Math.pow(10, halfLength);
        long b = x % (long) Math.pow(10, halfLength);
        long c = y / (long) Math.pow(10, halfLength);
        long d = y % (long) Math.pow(10, halfLength);

        // 3 recursive calls to numbers approximately half the size
        long z0 = karatsuba(a, c); // ac
        long z1 = karatsuba(a + b, c + d); // (a+b)(c+d)
        long z2 = karatsuba(b, d); // bd

        return z0 * (long) Math.pow(10, 2 * halfLength) + 
               (z1 - z0 - z2) * (long) Math.pow(10, halfLength) + 
               z2;
    }

    public static void main(String[] args) {
        long num1 = 1234;
        long num2 = 5678;
        System.out.println("Product: " + karatsuba(num1, num2));
    }
}
```
Output:
```
Product: 7006652
```
This implementation of the Karatsuba algorithm efficiently multiplies two large numbers by recursively breaking them down into smaller parts and combining the results.
