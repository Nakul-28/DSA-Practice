## Print Name n times

```java
class Main{
    public static void main(String[] args){
        int n = n; // any number or make this an input variable
        f(1,n);
    }
    public static void f(int i, int n){
        if(i>n){
            return;
        }
        System.out.println("ABC");
        f(i+1,n);
    }
}
```

## Print linearly from 1 to n

```java
class Main{
    public static void main(String[] args){
        int n = n; // any number or make this an input variable
        f(1,n);
    }
    public static void f(int i, int n){

            return;

        System.out.println(i);
        f(i+1,n);
    }
}
```

## Print linearly from n to 1

```java
class Main{
    public static void main(String[] args){
        int n = n; // any number or make this an input variable
        f(n,n);
    }
    public static void f(int i, int n){
        if(i==0){ // if(i<1) can also be used
            return;
        }
        System.out.println(i);
        f(i-1,n);
    }
}
```
