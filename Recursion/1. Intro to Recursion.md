- **Recursion:** When a function calls itself until a specified condition is met.

```java
class Main{
    public static void f(){
        System.out.println(1);
        f();
    }
    public static void main(String[] args){
        f();
    }
}
```

- This function will indefinitely call itself until it causes stack overflow error and crashes the program.

> A **segmentation fault** is an error that occurs when a program attempts to access a memory location that it is not allowed to access. In pure Java code, segmentation faults are **generally avoided by the design of the language and the Java Virtual Machine (JVM)**, though they can still occur in specific, rare circumstances.

> ### What is a Segmentation Fault?
> A segmentation fault (or "segfault") is a hardware-level error where the operating system terminates a running program because of an illegal memory access. This mechanism is a form of memory protection that prevents a faulty program from corrupting the memory of other running processes or the operating system itself.

## Recursion Tree
It is a pictorial representation used to analyze the time complexity of recursive algorithms. Each node in the tree represents the cost (time or work) of a single subproblem or recursive call, and the entire tree visualizes the breakdown of the original problem into smaller subproblems.
